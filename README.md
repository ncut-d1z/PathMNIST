# 医学图像分类

## 课题

基于PathMNIST数据集的ResNet与ViT模型训练

## 数据集

PathMNIST组织病理学图像数据集
    - 9个类别
    - 图像尺寸：28×28像素（RGB）

## 建模要求

### 数据预处理与分析

    - 加载并可视化数据集
    - 数据标准化
    - 分析类别分布

### ResNet 模型实现

实现 ResNet-18 或 ResNet-34

### Vision Transformer 实现

实现ViT模型

### 模型比较

    - 测试集评估两个模型
    - 计算准确率、精确率、召回率、F1分数
    - 可视化混淆矩阵

## 性能要求
    - 基础要求：两个模型测试准确率均达到85%以上
    - 良好表现：达到90%以上
    - 优秀表现：达到93%以上
